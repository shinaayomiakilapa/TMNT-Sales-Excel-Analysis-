# TMNT-Sales-Excel-Analysis-
# 🛍️ U.S. Retail Sales Performance Analysis
 
A comprehensive Excel-based sales analytics project analyzing retail performance across the United States — covering revenue trends, product categories, regional performance, sales team rankings, and the relationship between store demographics and revenue.
 
---
 
## 📁 Project Structure
 
The workbook (`Project_2.xlsx`) contains **5 sheets**:
 
| Sheet | Description |
|---|---|
| `Dashboard` | Visual summary of all key metrics |
| `Report` | Pivot-based analysis and KPI summaries |
| `Sales table` | Raw transactional data (1,000+ orders) |
| `Sales Team` | Sales representative and region mapping |
| `Products` | Product catalog with category classifications |
| `Store Locations` | Geographic and demographic data for 367 U.S. cities |
 
---
 
## 📊 Dataset Overview
 
### Sales Transactions (`Sales table`)
- **1,000+ order records** spanning a 5-month period (January–May)
- Each record includes: Order Number, Sales Date, Sales Channel, Currency, Sales Team, Store, Product, Order Quantity, Unit Price, Unit Cost, Revenue, and Profit
### Sales Channels
- Online
- In-Store
- Wholesale
- Distributor
### Geographic Coverage
- **44 U.S. states** represented across **367 store locations**
- Includes city-level demographic data: population, households, and median income

## 📂 How to Use
 
1. Open `Project_2.xlsx` in Microsoft Excel (2016 or later recommended)
2. Start on the **Dashboard** sheet for a high-level summary
3. Navigate to the **Report** sheet for detailed pivot analysis
4. Use the **Sales table** sheet to explore or filter raw data
5. Reference **Products**, **Sales Team**, and **Store Locations** sheets for dimension context
